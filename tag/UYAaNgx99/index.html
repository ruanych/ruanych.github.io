Windows下 更改 pip默认缓存目录

2020-09-20 Windows


windows下，pip的默认缓存目录为："C:\Users{username}\AppData\Local\pip\cache"


可以使用以下命令修改缓存目录 
pip config set global.cache-dir "D:\\Data\\Temp\\pip\\cache"
