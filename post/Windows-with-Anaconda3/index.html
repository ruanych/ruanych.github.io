Windows 下 Anaconda3 的安装配置


Anaconda指的是一个开源的Python发行版本，其包含了conda、Python等180多个科学包及其依赖项。



Anaconda3

从 官网 下载安装包



安装过程中值得注意的两点：

  • 安装路径别包含中文，这样避免了字符编码等问题产生的BUG。
  • 不要把 Anaconda3 加入环境变量，这样不会影响到系统环境变量。






建议使用 Anaconda3 管理 Python，不需要另外安装 Python。




新建工作空间

我们可以新建一个工作空间，避免对 root 工作空间（根目录）的影响。

启动 Anaconda Navigator， 在 Environments 中 Create 一个新的工作空间



单击新建好的工作空间进行切换，可以搜索下载自己需要的相关的包





安装 Jupyter notebook 和 Spyder

在 Home 中安装 Jupyter notebook 和 Spyder




PS: 修改 Anaconda3 jupyter 默认启动目录 点击此处
