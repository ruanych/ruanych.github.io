Linux 下搭建 Java Web 开发环境

2020-08-11 Linux


Java Web，是用Java技术来解决相关web互联网领域的技术栈，下方为 Linux 下搭建 Java Web 开发环境教程。



安装JDK

# Centos 使用yum安装JDK1.8
yum -y install java-1.8.0-openjdk*

# Ubuntu 使用apt-get安装JDK1.8
sudo apt-get install openjdk-8-jdk

# 检查是否安装成功
java -version



安装MySQL数据库

# Centos 使用yum安装MySQL
wget http://dev.mysql.com/get/mysql57-community-release-el7-10.noarch.rpm
yum -y install mysql57-community-release-el7-10.noarch.rpm
yum -y install mysql-community-server

# Ubuntu 使用apt-get安装MySQL
sudo apt-get install mysql-server



安装Tomcat

# 使用镜像下载Tomcat压缩包（可将链接中版本号换成最新版本号）
wget https://mirror.bit.edu.cn/apache/tomcat/tomcat-9/v9.0.37/bin/apache-tomcat-9.0.37.tar.gz

# 解压压缩包
tar -zxvf apache-tomcat-9.0.37.tar.gz 

# 将文件移动到 /usr/local 目录下
mv apache-tomcat-9.0.37 /usr/local/Tomcat9.0

# 为Tomcat授权
chmod +x /usr/local/Tomcat9.0/bin/*.sh

# 修改Tomcat默认端口号8080为80
sed -i 's/Connector port="8080"/Connector port="80"/' /usr/local/Tomcat9.0/conf/server.xml

# 启动Tomcat
/usr/local/Tomcat9.0/bin/./startup.sh



访问Tomcat

在浏览器中输入ECS公网地址，出现以下页面表示Tomcat安装配置成功



至此，Java Web开发环境搭建完成。
