Windows 下 修改 Anaconda3 jupyter 默认启动目录


Jupyter Notebook 是一个交互式笔记本，本质是一个 Web 应用程序，支持运行 40 多种编程语言，支持实时代码，数学方程，可视化和 markdown。

安装好 Jupyter 后的默认启动目录是系统盘的 用户目录，可以通过下面的方式修改启动目录。



  1. 启动cmd,执行以下命令，生成 jupyter 默认配置文件
jupyter notebook --generate-config

文件路径是 C:\Users\USERNAME\.jupyter，USERNAME 是你的系统用户名

如果之前没有生成配置文件，则直接生成，如果之前有，则可选覆盖



  1. 根据路径找到并打开 jupyter_notebook_config.py 配置文件，然后定位到 c.NotebookApp.notebook_dir 设置所在的一行



  1. 去掉该行前面的注释，并在后面写上 你期望的 jupyter 启动目录，保存文件

此时可以通过 Anaconda Navigator 打开 jupyter，工作目录是自定义的目录。



如果希望通过快捷方式打开，则进行下面的更改。



  1. 找到 jupyter 快捷启动程序



  1. 修改其属性（建议先备份该文件），将 目标 最后面的 双引号以及双引号里面的内容 删掉，只保留 ...... jupyter-notebook-script.py 前面的部分

    root 工作环境是删掉 "%USERPROFILE%"



之后通过这个快捷方式即可访问自定义工作目录的 jupyter 了。
